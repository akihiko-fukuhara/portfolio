KnynK(ナレッジシェアアプリ)

アプリ名 KnynK
コンセプト ローカルなコミュニティをターゲットにした yahoo 知恵袋。
機能 アカウント登録機能、ログイン機能、質問.情報共有の投稿機能、投稿に対しての回答機能、 回答に対してのコメント機能、グループ機能、BGM再生.停止、効果音。
使用方法
1. 任意の名前とパスワードを入力し、アカウントを登録。
2. 作成したアカウントの名前とパスワードを入力してログイン。
3. 画面右上の一番右側にはログイン中のアカウント名が表示される。
4. 画面右上の[投稿する]ボタンを押し、投稿画面から質問.情報共有ができる。
5. 画面右上の[グループ作成]ボタンを押し、グループ作成画面からグループの作成とアカウントの招待ができる。
6. 画面左側からグループを選択できる。デフォルトで全使用者共通のpublicグループがある。
7. グループ選択中はグループ内の投稿一覧が表示され、投稿を選択すると詳細画面に移動する。
8. 詳細画面では、[回答]ボタンを押すと回答画面に移動し、投稿に対して回答ができる。
9. 詳細画面に回答が表示されている時、その回答に対してコメントの入力欄からコメントできる。
10. 詳細画面の投稿と回答には[🐘]ピンクのボタンがあり、押下すると効果音が流れる。画面上部の[▶︎]再生ボタンを押すとBGMが流れる。
工夫点 グループ招待画面でアカウント名入力欄のどの欄に名前を入れても全員登録されるようにロジックを工夫した。
言語 Java-16
その他環境 サーブレット、JSP、Apache-Tomcat-9
DB PostgreSQL-14

家計簿アプリ

アプリ名 家計簿
コンセプト 直感的に分かりやすい家計簿。
機能 支出と収入の一覧表示、絞り込み検索、項目登録、項目編集、項目削除。
支出の月別グラフ表示。
収支の推移グラフ表示、月別一覧。
使用方法
1. 画面上部の項目から各画面へ推移する。
2. 支出一覧画面は、支出検索で要素毎に絞る範囲を選択する。支出一覧で絞り込まれた支出の全項目を表示する。また、操作欄の左のボタンで編集、右のボタンで削除できる。
3. 収入一覧画面は、支出検索で登録月で絞る範囲を選択する。収入一覧で絞り込まれた収入の全項目を表示する。また、操作欄の左のボタンで編集、右のボタンで削除できる。
4. 支出登録画面は、支出の登録ができる。
5. 収入登録画面は、収入の登録ができる。
6. 月間支出画面は、月毎の全支出の一覧とグラフが表示される。
7. 収支推移画面は、期間毎の収支推移がグラフに表示される。また、ラジオボタンの選択で絞り込みを行なった表示もできる。
8. 収支画面では、月毎の収入と支出の全項目と、その月の収入合計から支出合計をマイナスした金額が表示される。
9. 詳細画面に回答が表示されている時、その回答に対してコメントの入力欄からコメントできる。
10. 詳細画面の投稿と回答には[🐘]ピンクのボタンがあり、押下すると効果音が流れる。画面上部の[▶︎]再生ボタンを押すとBGMが流れる。
工夫点 グラフを表示して直感的に分かりやすくした。
言語 Python3.8
フレームワーク Django
DB Django Model
制作期間 8日
制作人数 1人

Instagramで使えるエフェクト

エフェクト名 Bear
機能 カメラで顔認識を行い、クマのエフェクトをかける。顔の向きや口の開閉に反応してエフェクトが動く。
使用方法
1. Instagramのストーリーズを開き、カメラを選択。
2. 画面下部でエフェエクトを選べるので一番右にスクロールし[🔍]項目を選択。
3. また[🔍]項目を選択し、「TestBear」と入力すると該当のものが出てくるので選択し使用できる。
ツール Spark AR Studio
業務形態 業務委託
業務内容 Instagramエフェクトの制作
制作期間 4日
制作人数 1人
